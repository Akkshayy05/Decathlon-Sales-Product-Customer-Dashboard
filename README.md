# Decathlon-Sales-Product-Customer-Dashboard
This project showcases an end-to-end Power BI Business Intelligence dashboard designed to analyze Decathlon’s sales performance across key business metrics. The report delivers actionable insights through interactive visuals, custom navigation, and dark-mode UI.

# Objectives

Build an easy-to-navigate, visually appealing BI dashboard.

Analyze Revenue, Orders, Profit, and Return Rate.

Drill down into category and product-level performance.

Demonstrate strong Power BI development, DAX, and ETL skills.

# Tools & Technologies Used

Power BI Desktop

Power Query (M Language) – for cleaning & transforming data

DAX – for measures & calculations

Data Modeling (Star Schema)

Power BI Navigation Buttons

GitHub (Project Documentation & Version Control)

# How to Use

Download or clone this repository

Open Dashboard.pbix in Power BI Desktop

Use the left-side navigation panel to explore report pages
Left-Side Navigation Button System (Custom Built)
The dashboard includes a custom-designed vertical navigation panel on the left side.
These icons/buttons are created to help users:
Navigate between pages (Home, Map, Product Detail, Customer Detail, Tooltip pages)
Move seamlessly without using Power BI’s default tabs
Maintain a clean, modern layout by hiding the standard page bar
Improve user experience through a consistent navigation system

Interact with slicers, tooltips, and visuals
